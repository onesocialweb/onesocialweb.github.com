Developers overview

Why onesocialweb?

Social networks are becoming an increasingly important part of our daily communication. Yet they are closed environments that do not allow you to friend, message or share across networks.

We dream of a world where all social networks are connected and work together in One Social Web. The OneSocialWeb projects aims to define the language to bridge these networks and make it easy for social networks to join.

What makes our approach unique is that we use XMPP as the engine. Please check the Why use XMPP? section why we picked this.

If you are excited about this like us, you can do several things to help:

  • For client developers: get the source code and play with our Java libraries and clients for Web and Android. Not into Java? Create an implementation of your choice using our protocol once it is stable.
  • For server developers: get the source code and play with our OpenFire plugin. Not into Java or OpenFire? Create an implementation or plugin for your server of choice using our protocol once it is stable.
  • Download the OpenFire plugin, setup your own server and join the federation.
  • Share your thoughts on the mailinglist.
